## 🎯 Что добавлено

### 1. Command Palette (⌘ K) - Быстрая навигация

**Как использовать:**
- Нажми `Ctrl+K` (или `Cmd+K` на Mac)
- Или кликни кнопку "⌘ K" в навигации
- Начни печатать название раздела
- Используй стрелки ↑↓ для навигации
- Enter для перехода
- Escape для закрытия

**Что можно найти:**
- 🏠 Все основные разделы (Главная, Ум, Тело, Контроль, Трекеры)
- 📚 Все подразделы (Библиотека, Шахматы, Тренировки, Бег, и т.д.)
- ▶️ Быстрые действия (Начать тренировку Push/Pull/Legs/HIFT)
- 🌙 Системные действия (Переключить тему, Экспорт данных)

**Поиск работает по:**
- Названию раздела
- Описанию
- Ключевым словам (на русском и английском)

**Пример:**
- Печатаешь "push" → находит "Push тренировка"
- Печатаешь "книги" → находит "Библиотека"
- Печатаешь "сон" → находит "Сон и восстановление"

## 🚀 Как это работает

### Command Palette

**Технические детали:**
- 50+ предустановленных команд
- Фильтрация в реальном времени
- Поддержка клавиатурной навигации
- Fuzzy search по всем полям
- Backdrop blur эффект

**Горячие клавиши:**
- `Ctrl+K` / `Cmd+K` - Открыть
- `↑` / `↓` - Навигация
- `Enter` - Выполнить
- `Escape` - Закрыть

### Sticky Tabs

**Технические детали:**
- `position: sticky` с `top: 70px`
- Автоматическое добавление класса `.scrolled` при скролле
- Плавная анимация тени и границы
- Работает на всех вкладках автоматически
- Passive scroll listener для производительности

## 🎨 Стилизация

### Цветовая схема:

**Command Palette:**
- Фон: `rgba(0,0,0,.85)` с blur
- Панель: `var(--bg2)` с тенью
- Выбранный элемент: `rgba(0,212,255,.1)`
- Иконки: 1.4em размер

**Tabs:**
- Неактивная: `rgba(255,255,255,.04)`
- Hover: `rgba(0,212,255,.08)`
- Активная: `rgba(0,212,255,.12)` с тенью
- Иконка: 1.1em размер
- Разделитель группы: `1px solid var(--brd)`

## 🐛 Известные ограничения

1. Command Palette не индексирует динамический контент (тренировки, записи)
2. Sticky tabs могут перекрывать контент на очень маленьких экранах (<320px)
3. Fuzzy search работает только по точному совпадению подстроки
