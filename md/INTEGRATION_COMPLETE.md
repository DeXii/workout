   - Записать несколько тренировок → проверить рекомендации по весам
   - Тренироваться несколько дней подряд → проверить streak counter
   - Экспортировать данные → проверить CSV/JSON файлы

## 🎯 Известные ограничения

1. **Тепловая карта**: Использует встроенную реализацию в index.html (работает корректно)
2. **Уведомления**: Требуют разрешения пользователя и HTTPS (или localhost)
3. **Service Worker**: Требует перезагрузки страницы после первой установки
4. **Автобэкап**: Срабатывает раз в неделю автоматически
