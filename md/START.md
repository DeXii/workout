# 🚀 Как запустить Ayanokoji System v3.0

## ⚠️ ВАЖНО: Не открывайте index.html напрямую!

Из-за CORS политики браузеров, приложение нужно запускать через локальный сервер.

---

## 🎯 Быстрый старт

### Вариант 1: Двойной клик (Windows)
Просто запустите файл:
```
start-server.bat
```

Браузер откроется автоматически на `http://localhost:8080`

### Вариант 2: Командная строка
```bash
cd "C:\Users\DEXI\work\workout-3 — копия"
npx http-server -p 8080 -o
```

### Вариант 3: npm скрипт
```bash
npm start
```

---

## 🌐 Доступ к приложению

После запуска сервера, откройте в браузере:
- **Локально**: http://localhost:8080
- **В сети**: http://[ваш-IP]:8080

---

## 🛑 Остановка сервера

Нажмите `Ctrl + C` в терминале где запущен сервер.

## 🧪 Тестирование

```bash

npm test

# Проверить линтинг
npm run lint
```

## 📱 PWA установка

1. Откройте приложение в браузере (http://localhost:8080)
2. В Chrome: Меню → "Установить приложение"
3. В Edge: Меню → "Приложения" → "Установить это приложение"

После установки приложение будет работать offline!

## 🐛 Решение проблем

### Проблема: "Address already in use"
Порт 8080 занят. Используйте другой порт:
```bash
npx http-server -p 8081 -o
```

### Проблема: "Cannot find module"
Установите зависимости:
```bash
npm install
```

### Проблема: Бесконечная загрузка
1. Откройте консоль браузера (F12)
2. Проверьте ошибки
3. Убедитесь, что сервер запущен (не открывайте file://)

### Проблема: Firebase не подключается
Это нормально! Приложение работает offline с локальными данными.
Таймаут Firebase: 5 секунд, затем загрузка продолжается.
