**Вариант 1: Главное меню**
- Кликни "Приоритеты" в верхнем меню

**Вариант 2: Command Palette**
```
Ctrl+K
Печатай: "приоритеты"
Enter
```

**Вариант 3: Мобильная навигация**
- Нижнее меню → Приоритеты (🎯)

### Как работать с приоритетами:

1. **Начни с красных (1-5)** - без них система не работает
2. **Добавляй оранжевые (6-11)** - когда красные стали привычкой
3. **Зелёные (12-18)** - для долгосрочного роста
4. **Серые (19-28)** - используй по необходимости
5. **Не пытайся делать всё сразу** - фокус на топ-5

## 💡 ЗАЧЕМ НУЖНЫ ПРИОРИТЕТЫ

### Проблема:
- Слишком много разделов и вкладок
- Непонятно с чего начать
- Легко потеряться в функциях

### Решение:
- **Чёткая иерархия** - что важнее всего
- **Пошаговый подход** - начни с топ-5
- **Навигация** - где найти каждую задачу
- **Фокус** - не распыляйся на всё сразу

### Как использовать:
1. Открой раздел "Приоритеты"
2. Посмотри топ-5 красных задач
3. Сделай их привычкой (2-4 недели)
4. Добавь оранжевые задачи
5. Постепенно двигайся вниз по списку
