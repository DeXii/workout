## 🚀 Как использовать

### Command Palette (⌘ K)

**Открыть:**
- Нажми `Ctrl+K` (или `Cmd+K` на Mac)
- Или кликни кнопку "⌘ K" в навигации

**Навигация:**
- Печатай название раздела
- Стрелки ↑↓ для выбора
- Enter для перехода
- Escape для закрытия

**Примеры поиска:**
```
"push" → Push тренировка
"медитация" → Медитация
"книги" → Литература
"hrv" → HRV и Дыхание
"график" → График / Месячный календарь
"память" → Память (навык)
"сон" → Прогресс (если есть вкладка сна)
```

### Sticky Tabs

- 📌 Вкладки "прилипают" к верху при скролле
- 👀 Всегда видны при прокрутке
- 🎨 Группировка по смыслу
