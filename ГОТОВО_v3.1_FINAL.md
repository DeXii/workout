## 🚀 КАК ИСПОЛЬЗОВАТЬ

### Command Palette (Ctrl+K)

**Примеры поиска:**
```
"push" → Push тренировка
"медитация" → Медитация
"завтрак" → Завтраки
"hrv" → HRV и Дыхание
"график" → График
"память" → Память (навык)
"записать" → Записать тренировку / Записать бег
"рекорды" → Личные рекорды
"фаза" → Фаза 0/1/2/3
```

**Навигация:**
- `Ctrl+K` - открыть
- Печатай название
- `↑` `↓` - выбор
- `Enter` - переход
- `Escape` - закрыть

### Sticky Tabs
- Прокрути страницу вниз
- Вкладки "прилипнут" к верху
- Появится тень для лучшей видимости
